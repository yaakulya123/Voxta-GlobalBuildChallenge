We are participating in the Global Build Challenge, a strict 24-hour hackathon focused on building tools that improve human communication. Our project targets the "Bridge the Gap" track, which asks developers to create accessibility layers that make video calls more inclusive. The final deliverable must be a functional web prototype that we can record for a demo presentation.

We are building a specialized accessibility layer for virtual meetings, specifically designed to help the 430 million people globally with disabling hearing loss or speech impairments. The application is a web-based video call simulator that acts as an inclusive alternative to standard Zoom. It provides real-time, bi-directional translation between spoken verbal language and physical sign language gestures, entirely powered by local browser AI instead of human interpreters.
